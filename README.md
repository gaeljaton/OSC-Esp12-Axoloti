# OSC-Esp12-Axoloti
get open sound control (osc) message and transmit to Axoloti DSP through serial gpio (PA3/PA2)<br />
Wiring :<br />
ESP RX <-> TX PA3 Axoloti<br />
ESP TX <-> RX PA2 Axoloti<br />
ESP 3v3 <-> VDD (any) Axoloti<br />
ESP Gnd <-> GND (any) PA2 Axoloti<br />
